How to Use

Using Connect is simple and straightforward. Follow these steps to get started:



  1. Access the Connect website by opening your web browser and navigating to the provided URL.
  2. Explore the navigation bar at the top of the page to learn more about the different sections of the website, including Home, About Us, and How to Use.
  3. On the Home page, you'll find a section titled "Select Chat Type." Choose between starting a Video Chat or a Text Chat by clicking on the corresponding button.
  4. If you choose to start a Video Chat, you'll be directed to a video chat room where you can interact with other users via video conferencing. Make sure to allow access to your camera and microphone if prompted.
  5. If you prefer a Text Chat, clicking on the Text Chat button will lead you to a chat room where you can communicate with other users via text messages. Simply type your message in the chat input box and press Enter to send.

That's it! You're now ready to start connecting with others on Connect. Enjoy your experience!
